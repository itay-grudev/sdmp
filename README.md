# SDMP Reference Implementation

Secure Device Management Protocol Client, Server and Device Client reference
implementation.

## Requirements
    * `odb`
    * `gnutls`
    * GNU `getopt` (with support for `getopt_long` available in the GNU implementation)

## Building

```bash
cmake ./
make
```
